How to use AVUS

Welcome to AVUS! Our system is designed to help you analyze text using advanced detection models. Follow these steps to make the most out of your experience:

Step 1: Input Your Text

Begin by providing the text you want to analyze. The system accepts texts between 250 and 5000 characters. If your text exceeds this size, you can choose to upload a PDF, DOCX, or TXT file instead.

Note: Please be cautious when uploading PDF and DOCX files, as the conversion process may sometimes lead to unexpected formatting issues.

Step 2: Process the Text

Once you've entered your text or uploaded a file, click the "Process Text" button. The system will perform the following steps:

  • Clean the text to remove any unnecessary formatting or special characters.
  • Analyze the text using three different detection models to provide a comprehensive assessment.

Step 3: Review the Results

After the analysis is complete, you will see the results displayed on the page. The results are divided into two main sections:

Overall Result

This section provides a summary of the combined analysis results. It gives you a quick overview of the text's characteristics.

Model Breakdown

In this section, you'll find a detailed breakdown of the analysis performed by each detection model. This breakdown offers insights into how different aspects of the text were evaluated.

Step 4: Line-by-Line Analysis

For an even deeper understanding, you can explore the line-by-line analysis of the text. This feature allows you to see how each line is assessed by the detection models. To access this analysis:

  • Click on the "Line-by-Line Analysis" section.
  • Each line of the text will be listed, and you can expand each line to view the corresponding analysis.

Step 5: Testing with Generated Text

If you'd like to test the system using generated text from GPT models, you have that option as well. Keep in mind:

  • For GPT-4 testing, you'll need to input an API key.
  • If you encounter any errors while testing, it's advisable to use your own OpenAI API key.

Enjoy using our text analysis tool and gaining insights into the characteristics of the text you provide!
